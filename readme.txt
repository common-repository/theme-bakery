A simple tool that allows you to generate a new blank theme (uses _S theme).

== Description ==

This plugin enables you to generate a new theme, based on the <a href="https://github.com/Automattic/_s">_S</a> theme.


== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Settings -> Theme Bakery and bake a new theme.
4. Enjoy!
